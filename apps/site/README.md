## ILATE Main Site

This folder is the consolidated recovery base for the main `ilate.pt` site.

Contents:

- `index.html`
  Main hub snapshot based on `index.live.patched.html`
- `.items_data.json`
  Local data snapshot used by the hub in static mode
- `workflow.html`
  Workflow view snapshot
- `orcs.html`
  ORCS live page snapshot

This is intended as a site-level recovery point inside the repo.
It is not yet wired to a dedicated production deploy workflow.
